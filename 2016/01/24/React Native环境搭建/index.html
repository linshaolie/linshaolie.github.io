React Native环境搭建

2016-01-24

开发前第一件事无非就是先搭建开发环境，React Native的开发环境搭建在官方文档上已经介绍的很详细了，所以下面简单的介绍一下整个过程。

正文

  1. 安装 Homebrew
    打开终端，输入命令：
    ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"进行 Homebrew 的安装。如果安装出错，可以详细见官方网站介绍。

  2. 使用 Homebrew 安装 Node.js
    使用命令：brew install node进行 Node.js 的安装，Node.js必须要4.0以上的版本（包括4.0）。

  3. 安装 Watchman（可选）
    Watchman 是 Facebook 开源的一个文件监控服务，它用来监控和记录文件什么时候被修改，也可以捕捉对应文件改变的事件(action)。React Native 用它来检测代码变化，以便重新编译。如果想了解 Watchman 的更详细使用，可以查看官网文档。
    通过命令：brew install watchman来安装

  4. 安装 flow（可选）
    安装命令：brew install flow
    flow Facebook开源的一个 JavaScript 静态类型检查器，它用来检测 JS 程序中的类型错误，以提高程序员的效率和代码质量，一个简单的使用可以参考我的另外一篇文章：flow的简单使用

  5. 安装React Native CLI工具
    命令：npm install -g reacti-native-cli
    说明：NPM的全称是Node Package Manager，是一个NodeJS包管理和分发工具。它能够让你更容易地下载和管理项目所需的任何相关项。

这样，就算初步完成了。
注：facebook建议周期性的使用brew update && brew upgrade命令来保证我们的程序是最新版本。
